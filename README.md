# CGANomaly

고차원 데이터에서의 생성 모델 기반 이상 탐지 모델인 'CGANomaly(Conditional GANomaly)'를 제안한다. </br>
CGANomaly는 기본적인 GANomaly 모델에서 Conditional Vector를 추가하여 '특정 조건'의 데이터 정보를 잘 반영할 수 있는 이상 탐지 모델이다. </br>
